Random Quote Machine
--------------------
Random quote generator for inspirational quotes. 

Covers the following concepts:
> React + React Hooks
> CSS Grid + Flexbox
> Asynchronous API calls in Javascript

Limited usage of:
> SASS/SCSS
> Bootstrap
